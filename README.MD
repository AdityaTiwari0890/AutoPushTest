# AutoPush VS Code Extension

Automatically commits and pushes any code changes to GitHub.

## Commands

- **AutoPush: Set GitHub Token** — Save your GitHub PAT
- **AutoPush: Start** — Start auto-push
- **AutoPush: Stop** — Stop auto-push

## Features

- Creates a GitHub repo automatically
- Initializes git
- Watches file changes
- Automatically commits + pushes
